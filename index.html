Linux指令的学习

目录控制

  1. cd   xx    —切换目录
  2. cd   ..    —返回上一级目录
  3. cd   ~    —代表HOME目录
  4. cd   -    —代表前一目录
  5. mkdir       目录名    —创建目录
  6. touch   文件名.格式    —创建文件，后面得带文件格式
  7. touch   目录/文件名.格式    —在指定目录下创建文件
  8. rm -f   文件名    —删除文件
  9. rm -rf   文件夹名    —删除文件或者文件夹(强制)

gcc编译C语言文件

  1. gcc   hello.cpp    —默认产生一个a.out文件

  2. gcc   -o   Hello   hello.cpp    —产生一个hello可执行文件
    PS:↓
    (hello.cpp是要编译的文件，Hello是生成编译后文件的名字)

  3. gcc   hello.c   -o   myhello    —产生一个myhello可执行文件 PS:↓
    (同样的道理,myhello就是生成编译后文件的名字)

  4. 运行gcc编译好的文件—-以书面编译好的Hello文件为例

    PS↓:输入 ./Hello 这个Hello就是文件编译后生成的文件的名字

  5. sudo su —开启超级用户

  6. su 用户名 —变为普通用户
